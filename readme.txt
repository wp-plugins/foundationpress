Foundation Press offers the Foundation Framework styles with WordPress Shortcodes. It makes adding styles (Tabs, Buttons, Grid...) pretty easy.

== Description ==

[FoundationPress](http://costartpress.com) offers the [Foundation Framework](http://foundation.zurb.com) styles with WordPress Shortcodes. The aim is to make adding elements such as Tabs, Grid, Buttons...) to your WordPress blog as simple as possible.

FoundaionPress support the following styles

= Typography =

* H1 to H6

* Keystrokes
* Blockquote
* Labels
* Buttons
* Panels

= Tables =

* Table Container
* Table Header and Body
* Table Body and Cell

= Grid =

* Grid Container
* 12 Column divisions

= Components =

* Tabs
* Accordion (Collapsible Menus)
* Tooltips
* Alerts

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open 'index.html' in the 'help' folder. Read the documentation for the shortcodes you want to use.
